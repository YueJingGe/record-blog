## 启动项目

1. 进入blog01文件夹下，安装依赖
```bash
npm i --save
```

2. 进入themes文件夹下，克隆主题
```bash
git clone https://github.com/Mrminfive/hexo-theme-skapp.git
```

3. 进入blog01文件夹下，启动

```bash
hexo server
````
